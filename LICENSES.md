# Licensing and provenance decision record

No repository-wide license has been selected, and this file is not a license
grant. Contributors must not infer a root license or add a `LICENSE` file
without an explicit owner decision.

The `contracts/cw-reality/Cargo.toml` package metadata declares Apache-2.0 and
its README records a clean-room reimplementation claim. The historical UI
README previously records AGPL-3.0 provenance. Those scoped historical
statements do not establish licensing for this repository as a whole.

Until owners resolve provenance and add the corresponding authoritative license
texts, copying, redistribution, and dependency-license conclusions require
manual legal review. Automated dependency checks are advisory and must not be
represented as resolving this open decision.
